## 查找组件文档

直接读取 `references/docs/组件名.md` 获取完整 API 文档。

示例: 查找 DForm 文档

```
Read: references/docs/DForm.md
```

示例: 搜索某个 API

```
Grep: "DTableProps" include: "*.md" path: "references/docs"
```
